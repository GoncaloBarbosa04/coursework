# InsideAirbnb Data Visualization Interface

> **Course:** Interação Pessoa-Máquina (Human-Computer Interaction) | **Year:** 2025/2026

## About this Project

Este projeto consiste no desenho e implementação de uma interface web para um sistema de visualização de dados focado na plataforma **InsideAirbnb**. O objetivo central da aplicação é facilitar a extração, análise e interpretação de dados complexos sobre o impacto do Airbnb (listagens, preços e ocupação) nas comunidades residenciais.

A interface foi desenhada com uma forte abordagem de _User-Centered Design_, garantindo a entrega de valor a três tipos de utilizadores distintos: investigadores académicos que necessitam de extração de dados complexos, gestores públicos focados em _dashboards_ de regularização, e ativistas que procuram gráficos claros e partilháveis.

## Core Competencies & Technical Skills Demonstrated

- **UI/UX Design & Prototyping:** Criação de protótipos interativos no **Figma**, com forte fundamentação teórica baseada em _guidelines_ de usabilidade e heurísticas.
- **Frontend Development:** Implementação completa da interface proposta utilizando a framework **Vue.js**, assegurando interatividade e consumo de APIs.
- **Data Visualization:** Conceção visual de _dashboards_ executivos, mapas por zonas da cidade, análise temporal e sistemas de alertas de anomalias (ex: propriedades com sobreocupação).
- **Requirements Engineering:** Levantamento de necessidades e adaptação de funcionalidades de interface com base em perfis de utilizador (_Personas_) estritamente definidos.

## Project Development Phases

### Phase 1: Prototyping & Usability Analysis

A primeira etapa incidiu sobre a ideação e prototipagem visual da aplicação no Figma. Esta fase inclui um relatório detalhado de usabilidade, demonstrando como a interface responde ativamente às necessidades de filtragem complexa, exportação standard (CSV/JSON) e criação de gráficos de impacto visual rápido.

### Phase 2: Frontend Implementation (Vue.js)

Tradução do protótipo validado numa aplicação _web_ real construída em Vue.js. O sistema interage de forma dinâmica com uma _Mockup API_, estruturando e apresentando os _datasets_ trimestrais de alojamentos, de forma a garantir uma navegação fluída, analítica e responsiva.
