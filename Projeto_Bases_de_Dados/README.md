# 📄 [Sistema de Base de Dados para a GreenMotion Solutions: Re-latório Técnico]

> **Course:** [Bases de Dados] | **Year:** [2025]

## 📌 About this Project

This project showcases a complete end-to-end database architecture lifecycle, from initial requirements gathering and conceptual modeling to logical design and final implementation. Developed for a sustainable mobility use case (GreenMotion Solutions), the primary challenge was designing a robust, scalable system capable of centralizing fragmented data across multiple sources (PostgreSQL, CSV, and JSON) into a unified MySQL platform.

By successfully resolving data redundancy, ensuring data consistency, and implementing secure architectural practices, this report demonstrates my ability to design efficient, production-ready database systems tailored to solve real-world operational bottlenecks and support future business scalability.

### 🛠️ Core Competencies Demonstrated:

- **Database Architecture & Design:** Conceptual and Logical Modeling.
- **Data Integration:** Consolidating disparate data sources (PostgreSQL, CSV, JSON).
- **Optimization:** Eliminating data redundancy and ensuring ACID compliance.
- **Tech Stack:** MySQL, Relational Database Management Systems (RDBMS).
